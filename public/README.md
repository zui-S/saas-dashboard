# Public Assets

## 文件说明

- `favicon.ico` - 网站图标
- `logo.svg` - Logo (珊瑚橙主题 #FF6B35)
- `og.png` - Open Graph 图片 (用于社交媒体分享)

## 颜色主题

- **主色**: Coral Orange `#FF6B35`
- **浅色**: `#FF8F66`
- **深色**: `#E85D2A`
- **黑色**: `#1A1A1A`
- **白色**: `#FFFFFF`

## 使用方式

```tsx
// 在组件中使用
<img src="/logo.svg" alt="Logo" />
<link rel="icon" href="/favicon.ico" />
```
